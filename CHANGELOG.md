# 📝 История изменений

## [1.1.0] - 2026-04-14

### ✨ Добавлено
- **Lucide Icons**: Заменены все эмодзи-иконки на профессиональные Lucide React icons
  - Статистика (Problem section): users, shield-alert, trending-down
  - Преимущества для IT-специалистов: graduation-cap, rocket, briefcase, handshake
  - Преимущества для стартапов: target, gem, wrench, user-plus
  - Форматы мероприятий: users, presentation, code, user-round-check, brain, calendar-days
  - Контакты: send (Telegram), mail (Email)

### 🔄 Изменено
- **Партнёры**: Обновлён список партнёров
  - Добавлен: **Пространство Просто** (стратегический партнёр)
  - Добавлен: **TOP IT COLLEGE** (образовательный партнёр)
  - Добавлен слот: "Ваша компания?" (призыв к партнёрству)
  - Удалены: Selectel, ФРИИ, Kama Flow

- **Первые 3 ивента**: Обновлены темы мероприятий
  - Ивент 1: Junior-to-Middle за год (площадка: Пространство Просто)
  - Ивент 2: Карьера в IT от студента до профессионала (партнёр: TOP IT COLLEGE)
  - Ивент 3: Нетворкинг и soft skills (площадка: Пространство Просто)

### 🎨 Улучшения дизайна
- Добавлены стили для Lucide icons во всех секциях
- Улучшена визуальная согласованность иконок
- Оптимизированы размеры и stroke-width для разных контекстов

### 🛠️ Технические изменения
- Подключена библиотека Lucide: `https://unpkg.com/lucide@latest`
- Добавлена инициализация `lucide.createIcons()` в script.js
- Обновлены CSS-стили для корректного отображения SVG-иконок
