# ⚡ Быстрый старт

## 3 шага до запуска

### 1️⃣ Подготовка (2 минуты)

```bash
# Переименуйте логотип
mv "!Лавлейс.jpg" logo.jpg

# Откройте index.html в браузере для проверки
```

### 2️⃣ Загрузка на GitHub (5 минут)

```bash
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/lovelace-club.git
git push -u origin main
```

### 3️⃣ Активация GitHub Pages (1 минута)

1. Откройте репозиторий на GitHub
2. Settings → Pages
3. Source: `main` branch, `/ (root)` folder
4. Save

**Готово!** Ваш сайт доступен по адресу:
```
https://YOUR_USERNAME.github.io/lovelace-club/
```
