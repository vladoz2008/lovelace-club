## 🆘 Troubleshooting

### Проблема: Логотип не отображается
- [ ] Проверить название файла: `logo.jpg`
- [ ] Проверить путь в HTML: `<img src="logo.jpg">`
- [ ] Проверить, что файл загружен в репозиторий
- [ ] Очистить кэш браузера (Ctrl+Shift+R)

### Проблема: GitHub Pages не работает
- [ ] Подождать 5-10 минут
- [ ] Проверить, что репозиторий Public
- [ ] Проверить настройки в Settings → Pages
- [ ] Проверить, что файл называется `index.html`

### Проблема: Форма не отправляется
- [ ] Открыть консоль браузера (F12)
- [ ] Проверить ошибки JavaScript
- [ ] Проверить токены/ключи в `script.js`
- [ ] Протестировать API отдельно

### Проблема: Анимации не работают
- [ ] Проверить подключение AOS.js
- [ ] Проверить консоль на ошибки
- [ ] Проверить атрибуты `data-aos` в HTML
