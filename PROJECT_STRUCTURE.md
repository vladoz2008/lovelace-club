## Секции лендинга (index.html)

### 1. Navigation (Навигация)
- Фиксированная шапка с логотипом
- Меню с якорными ссылками
- Адаптивное бургер-меню для мобильных

### 2. Hero (Главный экран)
- Крупный заголовок "Лавлейс"
- Описание миссии клуба
- 2 CTA-кнопки: "Вступить в клуб" + "Узнать больше"
- Анимированные плавающие фигуры на фоне
- Индикатор скролла

### 3. Problem (Проблема)
- Темная секция с контрастом
- 3 статистические карточки:
  - 500-700 тыс. дефицит кадров
  - 52% компаний без ИБ-кадров
  - 7,2 млрд ₽ венчурный рынок
- Описание вызовов IT-рынка

### 4. Solution (Решение)
- Светлая секция
- 2 колонки:
  - Для IT-специалистов (4 преимущества)
  - Для стартапов (4 преимущества)
- Описание бизнес-модели

### 5. Formats (Форматы мероприятий)
- Сетка из 6 карточек форматов:
  - Митап + нетворкинг
  - Startup Demo Day
  - Хакатон
  - Менторские пары
  - Мастермайнд-группы
  - Годовая конференция
- Блок "Первые 3 ивента" с темами

### 6. Roadmap (Дорожная карта)
- Интерактивная временная шкала
- 5 этапов развития (год 1 → год 5)
- Детализация первого года (3 фазы)
- Ключевые метрики на каждом этапе

### 7. Partners (Партнёры)
- 3 карточки партнёров:
  - Selectel (стратегический)
  - ФРИИ (стандартный)
  - Kama Flow (стандартный)
- Описание взаимной ценности

### 8. Contact (Контакты)
- Темная секция
- Описание следующего шага
- Ссылки на Telegram и Email
- Форма обратной связи:
  - Имя
  - Email
  - Роль (IT-специалист/Стартап/Партнёр)
  - Сообщение

### 9. Footer (Подвал)
- Логотип
- Копирайт "Санкт-Петербург | 2026"

## Стили (style.css)

### CSS Variables (Переменные)
```css
--color-bg: #F5EFE7          /* Бежевый фон */
--color-primary: #213547      /* Темно-синий */
--color-secondary: #4A90A4    /* Голубой */
--color-accent: #7EBDC3       /* Светло-голубой */
--color-peach: #F4A261        /* Персиковый акцент */
```

### Основные компоненты
- Reset & Base Styles
- Navigation (фиксированная, с эффектом при скролле)
- Buttons (primary, secondary)
- Hero Section (с анимированным фоном)
- Section Styles (общие стили для всех секций)
- Cards (stat-card, format-card, partner-card)
- Timeline (дорожная карта)
- Forms (контактная форма)
- Responsive Design (адаптив для mobile/tablet)

### Анимации
- `float` - плавающие фигуры
- `scroll` - индикатор скролла
- `rainbow` - пасхалка (Konami code)
- AOS animations - появление элементов при скролле

## JavaScript (script.js)

### Основной функционал

1. **AOS Initialization**
   - Инициализация библиотеки анимаций

2. **Mobile Menu**
   - Открытие/закрытие бургер-меню
   - Закрытие при клике на ссылку

3. **Navbar Effects**
   - Изменение тени при скролле
   - Подсветка активной секции

4. **Smooth Scroll**
   - Плавная прокрутка к якорям

5. **Form Handler**
   - Обработка отправки формы
   - Валидация (базовая)
   - Интеграция с Telegram/Email (настраивается)

6. **Parallax Effect**
   - Параллакс для плавающих фигур

7. **Easter Egg**
   - Konami code (↑↑↓↓←→←→BA)
   - Rainbow эффект

8. **Console Messages**
   - Приветственное сообщение
   - Подсказка про пасхалку
